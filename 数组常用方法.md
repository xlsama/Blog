### 判断数组中是否存在某个值，有则删除，没有则添加

值是字符串或者数值:

```js
const arr=[1,2,3]
let key=1
const newArr=arr.includes(key)?arr.filter(item=>item!==key):[...arr,key]
```

值是对象

```javascript
const arr=[
  {key:1},
  {key:2},
  {key:3}
]
const obj={key:2}
const newArr=arr.some(item=>item.key===obj.key)
			 ?arr.filter(item=>item.key!==obj.key)
			 :[...arr,obj]
console.log(newArr)// [{key:1},{key:3}]
```

### 在JS数组指定位置插入元素

```js
var arr=[1,2,4,5]
arr.splice(2,0,3)// [1, 2, 3, 4, 5]
```
